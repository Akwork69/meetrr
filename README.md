# meetrr

Random video chat application built with React, Vite, TypeScript, Tailwind CSS, and Supabase signaling.

## Local setup

```sh
npm install
npm run dev
```

## Build and test

```sh
npm run build
npm run lint
npm run test
```

## Environment

Create a `.env` file using `.env.example`:

- `VITE_SUPABASE_PROJECT_ID`
- `VITE_SUPABASE_PUBLISHABLE_KEY`
- `VITE_SUPABASE_URL`
